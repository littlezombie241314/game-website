# 🌍 全球飞行模拟器 - 5D地球探索游戏

## 游戏简介

这是一个基于WebGL技术的全球飞行模拟器游戏，让玩家能够驾驶酷炫的金属飞行器在世界各地自由飞行。游戏采用Three.js实现3D渲染，支持键盘和游戏手柄控制。

## 🎮 游戏特色

### 🌐 全球5D地图（CesiumJS集成版）
- **专业WebGL地球渲染**：集成CesiumJS库，提供电影级的地球可视化效果
- **真实地形数据**：使用全球地形数据，呈现山脉、海洋等真实地理特征
- **3D地形增强**：地形夸张效果、深度测试、光照渲染，让地形更加立体真实
- **动态城市标记**：基于SVG的交互式城市标记，支持点击选择和飞行导航
- **3D建筑模型**：主要城市标志性建筑的3D模型（东京塔、埃菲尔铁塔、帝国大厦等）
- **水面效果**：动态水面材质，包含波浪动画和反射效果
- **多视角切换**：支持地球视角、飞行视角和驾驶舱视角的无缝切换
- **高性能渲染**：利用WebGL技术实现流畅的3D地球浏览体验
- **精确城市定位**：基于经纬度坐标的全球城市标记系统
- **交互式全球地图界面**：点击选择飞行目的地（东京、巴黎、纽约、悉尼、迪拜）

### ✈️ 金属飞行器
- 酷炫的金属材质机身设计
- 真实的飞行物理引擎
- 驾驶舱内部视角切换
- 玻璃驾驶舱透明效果

- **键盘控制**: WASD键 + 方向键

### 📊 飞行仪表盘
- 实时高度显示
- 飞行速度监控
- 燃料消耗系统
- 位置信息追踪

## 🚀 游戏玩法

1. **选择目的地**: 在全球地图上点击想要飞行的城市
2. **外部视角**: 观察飞行器在空中的英姿
3. **进入驾驶舱**: 切换到第一人称视角体验真实飞行
4. **自由探索**: 使用控制设备在世界各地自由飞行

## 🎯 控制说明

### 键盘控制
- **W**: 上升
- **S**: 下降  
- **A**: 左转
- **D**: 右转
- **↑**: 加速
- **↓**: 减速

### 游戏手柄
- **左摇杆**: 控制飞行方向
- **右摇杆**: 控制视角
- **A/X按钮**: 进入驾驶舱
- **B/O按钮**: 返回地图
